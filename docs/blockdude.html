Block Dude is based on a program written by Brandon Sterner of Detached Solutions for the Texas Instruments family of graphic display calculators. The graphics and level data was pinched from the Z80 assembly code for the original game.

Use the arrow keys to move around, space to pick up or put down a block, the W A S D X keys to look around a level, R to reset a level or ESC to input a password to access a level.
